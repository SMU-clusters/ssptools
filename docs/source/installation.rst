============
Installation
============

SSPtools can be installed directly using pip, from PyPI:

.. code-block:: console

   $ pip install astro-ssptools

or from it's `Github page`_ directly:

.. code-block:: console

   $ pip install git+https://github.com/SMU-clusters/ssptools.git

.. _Github page: https://github.com/SMU-clusters/ssptools
